# Installation
### 🐧 **Linux Installation**
<pre>
<code>
git clone https://github.com/amir-rhmati/email-sender.git
cd email-sender
pip install -r requirements.txt
python sender_rahmati.py
</code>
</pre>


### 🐧 **Termux Installation**
<pre>
<code>
pkg update && pkg upgrade -y
pkg install git python -y
git clone https://github.com/amir-rhmati/email-sender.git
cd email-sender
pip install -r requirements.txt
python sender_rahmati.py
</code>
</pre>
